# 🍫 Chocolate Sales Analysis Dashboard (Power BI)

## Project Overview

This project presents an interactive Power BI dashboard for analyzing chocolate sales performance across multiple countries.

The dashboard provides insights into sales trends, product performance, shipment distribution, and profitability.

---

## Dashboard Features

- Current Year vs Last Year Sales
- Current Year vs Last Year Boxes Sold
- Sales by each Country
- Shipment Distribution
- Top 5 Salespersons by Profit
- Top 6 Products
- Product Contribution Percentage
- Interactive Filters and Slicers

---

## Tools Used

- Power BI Desktop
- Microsoft Excel
- DAX
- Power Query

---

## KPIs

- Total Sales
- Total Profit
- Total Boxes Sold
- Sales Growth
- Country-wise Sales
- Product Performance
